# 博客系统

> 一个初级vue + node练手项目

** 2018.05.15 ** 抽时间会写一个nuxt版本，并且巩固一下基础

### 技术栈

___

  * 前端
    * Vue
      * Vuex
      * vue-router
    * Webpack
    * Axios
    * Sass
    * MavonEditor
    * ElementUI
  
  * 中间层
    * Nginx(暂未使用反向代理)
    * Node
      * Koa
      * Koa-router
      * Koa-cors
      * Koa-bodyparser
      * Lodash
      * TrimHtml

  * 数据库
    * MongoDB
      * Mongoose
